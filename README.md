🔥 SecureLog Analyzer – CiberDefender Suite

Proyecto académico desarrollado como laboratorio práctico para análisis forense de registros en Java.

🎮 Descripción

Este proyecto forma parte de la CiberDefender Suite y permite cargar registros de seguridad desde un archivo .txt, clasificarlos según severidad y obtener una puntuación basada en el análisis.

Enfoque del proyecto: lógica, análisis, programación modular y diseño UI con estilo gamer/cyberpunk.
Archivo Log.txt incluido en  /lab2/test

🛠 Tecnologías

Java

Swing (UI)

Lectura de archivos (BufferedReader)

▶️ Ejecución

Abrir el proyecto en un IDE Java compatible (NetBeans, IntelliJ, Eclipse).

Ejecutar el archivo:

Main.java
